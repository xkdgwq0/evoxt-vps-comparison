## そもそも VPS ホスティングって何が違うの？

VPS（Virtual Private Server）は、物理サーバーを仮想化して複数のユーザーで分け合う仕組みだが、共有サーバーと違って各ユーザーに**専用のCPUコア・RAM・ストレージが割り当てられる**。

つまり「隣の人がうちのリソースを食べる」という現象が起きにくい。Webサイトが急に重くなる、アプリがたまにタイムアウトする——そういう症状に心当たりがあるなら、VPS ホスティングへの移行を検討する価値がある。

具体的な用途としてはこんな感じ：

- **Webサイト・ブログの高速化**（WordPressなど）
- **開発環境の構築**（Dockerコンテナ、Node.js、Pythonなど）
- **ゲームサーバー**（Minecraftなど）
- **VPN・プロキシの自前構築**
- **Discordボットの常時稼働**
- **Nextcloud等のセルフホスト型クラウドストレージ**

クラウドサービスに毎月お金を払うのがもったいない、という人にも VPS は人気がある。Dropboxの代わりにNextcloudを立てる、Notionの代わりにObsidianをセルフホストする——そういう使い方ができる。

## 管理機能・セキュリティ

VPS ホスティングを選ぶとき、デプロイした後の管理画面のわかりやすさも地味に重要なポイントだ。

Evoxtのコントロールパネルで使える主な機能：

- **モニタリング**：CPU・RAM・ディスクのリアルタイム使用状況
- **ファイアウォール**：レイヤー3のエンタープライズ級DDoS防御。SSH接続なしでルール設定が可能
- **VNCアクセス**：ブラウザからいつでもVMに直接アクセスできる
- **クローン機能**：同じ構成のVMを簡単に複製
- **週次自動バックアップ**：全プランで無料。万が一の際に数クリックで復元
- **IPアドレス管理**：IPの付け替えや追加が簡単にできる
- **サブアカウント**：チームごとに権限を分けて管理（開発チーム・課金担当など）
- **API**：コントロールパネルにログインしなくてもVMを操作できる

ファイアウォールがVM外側のネットワークレベルで動いているのは地味にうれしい。間違ってSSHのポートをふさいでしまっても、パネルから解除できる。
